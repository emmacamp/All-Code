Mongo DB

MongoDB (from hu mongo us , "huge") is an open source , document - oriented, NoSQL database system.

Instead of storing data in tables, as is done in relational databases , MongoDB stores BSON data structures (a specification similar to JSON ) with a dynamic schema, making data integration in certain applications easier. easier and faster.

MongoDB is a database suitable for use in production and with many features. This database is widely used in the industry, 1 with implementations in companies such as MTV Network, 2 Craigslist, 3 Foursquare. 4
